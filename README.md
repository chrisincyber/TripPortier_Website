# TripPortier Website

This is the official website for TripPortier, hosted on GitHub Pages at [tripportier.com](https://tripportier.com).

## About

TripPortier is a travel companion app for iOS that helps you pack smarter and travel better. The app includes AI-powered packing lists, flight tracking, eSIM data, and more.

## Structure

- `index.html` - Homepage
- `faq.html` - Frequently asked questions
- `feature-request.html` - Feature voting page
- `privacy.html` - Privacy policy
- `terms.html` - Terms of service
- `css/` - Stylesheets
- `js/` - JavaScript
- `assets/images/` - Logo, favicon, and screenshots

## Hosting

The site is deployed via GitHub Pages with a custom domain configured through the CNAME file.
