  1. Open this file in your browser (you're already here!)
  2. Take a screenshot:
    • Mac: Press Cmd + Shift + 4, then select just the blue gradient box
    • Windows: Use Snipping Tool or Win + Shift + S
  3. Save the screenshot as og-image.png
  4. Place it in your /assets/images/ folder
  5. Copy it and rename to twitter-card.png (Twitter uses same dimensions)
  6. Commit and push to GitHub!
